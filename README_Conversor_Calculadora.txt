
# Conversor de Moeda e Calculadora Científica

Bem-vindo ao projeto **Conversor de Moeda e Calculadora Científica**! Este é um aplicativo web que permite aos usuários converter valores entre diferentes moedas e realizar cálculos científicos básicos. Criado por mim, Clayton Souza, este projeto tem como objetivo facilitar a conversão de moedas e oferecer uma ferramenta prática para cálculos diretamente no navegador.

## Visão Geral

O projeto foi desenvolvido utilizando HTML, CSS e JavaScript, com o suporte de algumas bibliotecas externas para funcionalidades adicionais. A aplicação é responsiva e foi otimizada para funcionar tanto em dispositivos desktop quanto em dispositivos móveis.

## Funcionalidades

- **Conversor de Moeda:**
  - Conversão em tempo real entre várias moedas utilizando uma API de câmbio.
  - Interface intuitiva que permite selecionar a moeda de origem e destino.
  - Visualização clara dos valores convertidos com bandeiras dos países.

- **Calculadora Científica:**
  - Operações básicas como adição, subtração, multiplicação e divisão.
  - Funções científicas, incluindo raiz quadrada e cálculo de potências.
  - Design compacto para se ajustar ao layout da página.

- **Letreiro de Mercado Financeiro:**
  - Exibe informações atualizadas sobre o mercado financeiro, incluindo preços de ações selecionadas.

- **Design Responsivo:**
  - Layout otimizado para dispositivos móveis, ocultando elementos desnecessários para economizar espaço.
  - Adaptável a diferentes tamanhos de tela, incluindo iPhones e dispositivos Samsung Galaxy.

## Tecnologias Utilizadas

- **HTML5**: Estruturação da página.
- **CSS3**: Estilização e layout responsivo.
- **JavaScript**: Lógica da aplicação e interatividade.
- **jQuery**: Facilitar a manipulação do DOM.
- **Select2**: Melhorar a experiência do usuário na seleção de moedas.
- **Flag-Icon-CSS**: Exibir bandeiras de países no seletor de moedas.

## Instruções de Uso

1. **Clonar o Repositório:**
   ```bash
   git clone https://github.com/seu-usuario/conversor-moeda-calculadora.git
   ```

2. **Abrir o Projeto:**
   - Abra o arquivo `index.html` no seu navegador preferido.

3. **Utilizar o Conversor de Moeda:**
   - Selecione a moeda de origem no primeiro dropdown.
   - O Real Brasileiro está fixo como moeda de destino.
   - Insira o valor a ser convertido e clique em "Converter".

4. **Utilizar a Calculadora:**
   - Use os botões para realizar cálculos básicos ou científicos.
   - O display da calculadora exibe o resultado instantaneamente.
